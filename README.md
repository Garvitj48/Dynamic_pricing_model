# 💰 Dynamic Pricing Model for E-commerce

> A machine learning system that predicts the **optimal product price** to maximise sales volume — powered by Random Forest and deployed with Streamlit.

---

## Overview

E-commerce sellers constantly wrestle with pricing decisions: set the price too high and lose customers to competitors; set it too low and leave money on the table.  This project builds a **regression-based ML model** that analyses demand, seasonality, competitor pricing, and discount level to recommend the single price that will maximise predicted sales volume.

The end-to-end pipeline covers synthetic dataset generation, model training, evaluation, and a live interactive web app where you can query the model in real time.

---

## Features

- 🔢 **Synthetic dataset** — 10,000 realistic e-commerce records generated with a demand–price–sales formula
- 🌲 **Random Forest Regressor** — ensemble model with 100 trees, trained on 6 engineered features
- 📊 **Model evaluation** — MAE and R² reported after every training run
- 🔍 **Price sweep** — tests ₹199 → ₹999 in ₹10 steps and selects the sales-maximising price
- 🌐 **Streamlit web app** — interactive UI with metric cards and an explainability section
- 🧩 **Modular codebase** — separate scripts for data generation, training, and prediction

## Installation

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/dynamic-pricing-model.git
cd dynamic-pricing-model

# 2. (Recommended) create a virtual environment
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt
```

---

## Usage

### Step 1 — Generate the dataset
```bash
python mock_data.py
# ✅ Dataset saved to 'data/online_retail.csv'  (10,000 rows)
```

### Step 2 — Train the model
```bash
python src/train.py
# ── Model Performance ─────────────────────
#    MAE  : 26.43 units
#    R²   : 0.8791
# ✅ Model saved → models/pricing_model.pkl
```

### Step 3 — Launch the web app
```bash
streamlit run app.py
# Local URL:  http://localhost:8501
```

### (Optional) Run the prediction smoke-test
```bash
python src/predict.py
# Prints optimal price, predicted sales, and % lift for 3 test cases
```
